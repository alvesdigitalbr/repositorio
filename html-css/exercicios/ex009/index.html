Lista de definições

HTML
Linguagem de marcação para criar páginas web, isso é a habilidade de usar comandos/tag para deixar o site incrível
CSS
Linguagem de estilo para formatar páginas web, isso da o visual bonito para o site.
JavaScript
Linguagem de programação para adicionar interatividade e dinamismo às páginas web. isso é o que deixa o site com interações e profissionalismo
